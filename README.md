# Project management

Managing company's departments, project and tasks. The App works on both computer and mobile devices<br/>
This is the frontend repository.

Main techs are used in this app :

- React 18
- Vites
- Ant Design 5.0
- Redux Toolkit/ RTK Query
- SocketIO
